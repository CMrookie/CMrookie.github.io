python基础

  • Python
  • 学习笔记

注释

python中注释以“#”开头

变量

python中对变量的定义不需要什么额外的操作，直接写上需要声明的变量并赋值即可。但是声明时没有赋值会报错。

数据类型

  • Number
  • String
  • List
  • Tuple
  • Set
  • Dictionay

number

python中支持对整数和浮点数直接进行四则运算

string

对字符串的定义方式可以是' ',也可以是" "。

在定义的字符串中经常需要使用其他符号，如',",\这时可以在字符串中加上前缀r

需要多行字符串时可用''' '''表示，这就免去了拼接字符串的操作

list

list是一种有序的集合，可以方便的添加和删除其中的数据

tuple

tuple与 list相似，都是有序集合。但与 list不同的是，tuple一旦创建完就不能修改

set

set与 lis类似，是一组数据的集合，不同的是set是无序的，并且与dict的键不能重名一样，set的元素不能重复

dict

dict是一种无序的，以键值对的形式将数据存储于内存堆的数据

函数

python中的函数需要以def语句定义，依次写出函数名，括号，括号中的参数，冒号，最后在缩进块中编写函数

列表生成式

  • 生成列表

    使用range()方法可以方便的生成连续的的简单列表，但是如果需要生成的列表是这样的[1x1, 2x2, 3x3, ..., 10x10]，那么我们就需要编写较长的代码去实现

    此时列表生成式就可以派上用场

  • 复杂表达式
  • 条件过滤

    列表生成式的for循环后面可以加上if

  • 多层表达式

    列表表达式中也可以用多层for循环生成列表
